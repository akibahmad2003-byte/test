# Simple E-commerce demo

This repository contains a minimal ecommerce demo (backend in Express + SQLite, simple frontend) intended as a starter scaffold.

To run locally:

1. Install dependencies

   npm install

2. Start the server

   npm start

3. Open in your browser:

   http://localhost:3000

Notes:
- This is a demo. Sessions are stored in-memory and the checkout is simulated.
- For production you'd add authentication, persistent session store, payment integrations, validation, and tests.
